# Research — source of truth for Loop

Plain-markdown versions of the bootcamp research, so any design agent (Stitch, Claude, etc.) can
read and ground on it directly. The original graded files (`.docx` / `.pdf`) live in `deliverables/`;
these markdown files carry the same content in a readable form.

**Ground every design decision in these files. Do not invent context.**

| File | What it is |
|---|---|
| `00_product_reference.md` | Full product spec — what Loop is, the hero interaction, the six screens, how it stays distinct |
| `01_interviews.md` | Six user interviews (Mehdi O., Hamza I., Hamza jr., Youssef, Mehdi B., Mohammed) — the raw pain points |
| `02_personas.md` | Yassine (primary), Salma (mentor), Karim (visiting faculty) |
| `02_journey_map.md` | Six-stage current-state journey + pain points & opportunities + the key insight |
| `03_crazy8s_competitor_analysis.md` | Eight concepts + the competitor gap table |
| `04_pitch_kit.md` | One-line strategy, hero interaction, the 2-minute pitch mapped to the readings |

## The one-line problem
At AUI the answers that matter live in people's heads and leave when they graduate. Official channels
hold policy; peers hold practice; neither is durable, searchable, and trustworthy at once. Loop closes
that gap: AI drafts from AUI's own corpus, shows confidence + provenance, a verified senior confirms,
and the confirmation retrains the model.

## Design system (non-negotiable)
Forest green `#1D2B22` primary, warm sand `#FFF8EF` background, terracotta `#B5651D` the AI accent,
blue `#2C5F8A` = official policy, amber `#B5651D` = lived experience. Source Serif 4 headings, Work
Sans body. Terracotta owns every AI moment.
